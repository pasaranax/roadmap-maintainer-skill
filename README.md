# roadmap-maintainer skill

A Codex skill that keeps work aligned with a file-based roadmap and guides the agent through a self-check execution loop while completing tasks.

For first use, ask the agent:
`Download and install the skill from https://github.com/pasaranax/roadmap-maintainer-skill, then create a roadmap using the roadmap-maintainer skill.`

To create a new roadmap phase for a larger body of work, discuss it with the agent.

No extra instructions are needed to create slices inside a phase: the skill adds its trigger rule to `AGENTS.md` and then works automatically.
